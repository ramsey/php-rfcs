====== PHP RFC: __hashKey magic method ======
  * Version: 0.9
  * Date: 2014-10-22
  * Author: Joe Watkins
  * Status: Draft
  * First Published at: http://wiki.php.net/rfc/hashkey

===== Introduction =====
Often discussed is the problem that we cannot use objects as array keys.

===== Proposal =====
A new magic __hashKey method, expected to return a scalar suitable for use as an HashTable key.

===== Proposed PHP Version(s) =====
7
