
<h1 class="sectionedit1" id="request_for_comments">Request for Comments</h1>
<div class="level1">

<p>
This page gives an overview of the current RFCs for PHP.
</p>

<p>
To create a new <abbr title="Request for Comments">RFC</abbr>, see <a href="/rfc/howto" class="wikilink1" title="rfc:howto" data-wiki-id="rfc:howto">How To Create an RFC</a>.
</p>

<p>
Note: An <abbr title="Request for Comments">RFC</abbr> is effectively “owned” by the person that created it. If you want to make changes, get permission from the creator. If no agreement can be found, the only course of action is to create a competing <abbr title="Request for Comments">RFC</abbr>. In this case, the old <abbr title="Request for Comments">RFC</abbr> page should be modified to become an intermediate page that points to all the competing <abbr title="Request for Comments">RFC</abbr>&#039;s.
</p>

<p>
A new page in this <abbr title="Request for Comments">RFC</abbr> namespace will automatically be loaded with an <abbr title="Request for Comments">RFC</abbr> <a href="/rfc/template" class="wikilink1" title="rfc:template" data-wiki-id="rfc:template">template</a>. Customize as needed.
</p>
